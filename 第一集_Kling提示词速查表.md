## 使用说明

**提示词结构**（每条遵循此顺序）：
```
[场景素材引用] [主体动作与表情] [镜头运动指令] [光线与色调] [氛围情绪]
```

**参考素材引用格式**：
- `参考素材` 栏注明应导入的GPT定稿图资产代码
- 首帧参考图建议上传对应场景或人物的定稿图
- 系统UI特效镜头标注`[后期AE合成]`，Kling只生成人物动作层

**Kling 3.0 运镜关键词（直接使用）**：
| 中文指令 | 效果描述 |
|----------|----------|
| 固定镜头 | 静止机位，无摇移 |
| 缓慢推镜 | 镜头匀速向主体推进（zoom in） |
| 缓慢拉镜 | 镜头匀速从主体拉远（zoom out） |
| 向右横摇 / 向左横摇 | 水平摇镜（pan） |
| 低角度仰拍 | 机位低于主体，仰视构图 |
| 过肩跟随 | 从被摄主体肩后跟随前行 |
| 手持抖动 | 模拟手持摄影机轻微震动 |
| 向上升镜 / 向下降镜 | 镜头垂直移动 |
| 缓慢下摇 / 缓慢上摇 | 镜头沿垂直轴缓摇 |
