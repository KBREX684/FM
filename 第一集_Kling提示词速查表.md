# 第一集 · Kling提示词速查表

> **使用说明**：
> - 每个镜头提供**可直接复制粘贴**的Kling提示词
> - 提示词 = 全局基底（见`全局美术资产/画风规范/全局Kling画风提示词.md`）+ 本镜专属内容
> - 标注 🎬 = Kling负责生成的内容
> - 标注 ✂️ = 后期合成，Kling不处理
> - 每镜提示词直接复制"【Kling提示词】"框内文字即可

---

## 一、如何使用

```
第一步：复制本文件对应镜头的【Kling提示词】
第二步：在Kling中设置参数（时长/运镜方式）
第三步：生成后与【一致性核查点】逐项对比
第四步：后期阶段按【后期制作清单】补充特效
```

# 附录B：三个不能直接用Kling生成的内容说明

### ① 漫画文字特效（S01-006 / S02-005 / S03-002 / S03-008）
Kling无法生成精确的中文漫画字体弹出效果，必须后期在AE/剪映等软件中叠加。
- Kling只生成角色底图（人物表情/动作）
- 后期叠加：文字+运动路径+音效

### ② 系统面板P04（S03-015 / S03-017）
面板是独立的UI资产，需要专门设计+动画制作，与Kling生成的人物/背景底图合成。
- Kling生成角色+背景底图
- 面板独立制作（UI设计+After Effects动画）
- 最终AE合成

### ③ 唇印特写（S03-013 子镜A）
Kling对于细节道具特写的唇印颜色控制不稳定，建议：
- 先用Kling生成瓶口特写
- 如唇印不清晰/颜色不对，用绘图软件手动补绘唇印层
