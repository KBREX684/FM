# 全局 Kling 画风规范 · 提示词基底

> 本文件定义所有集数生成视频时的**全局风格提示词基底**。  
> 每个镜头提示词必须在此基底之上叠加镜头专属内容。  
> 最终提示词 = **全局基底** + **镜头专属内容**

---

## 一、正向提示词基底（必须包含）

```
【风格基底-中文版】

色块为主加柔和阴影，介于韩漫扁平与日系赛璐珞之间，
7~7.5头身比例，竖屏9:16构图，电影级画面质量，
高帧率流畅动画，无水印，高分辨率

【风格基底-英文版（可提升生成质量）】
Chinese urban comic animation style, semi-rendered coloring, 
clean linework with hand-drawn feel, 9:16 vertical format, 
cinematic quality, smooth animation, no watermark, 
high detail, 7-head proportions
```

## 六、统一技术规格

| 参数 | 规格 |
|------|------|
| **画面比例** | 9:16 竖屏 |
| **风格定位** | 国漫轻写实，半渲染 |
| **角色比例** | 7-7.5头身 |
| **线条风格** | 清晰但不硬朗，保留手绘感 |
| **上色风格** | 色块为主+柔和阴影，避免厚涂 |
| **场景饱和度** | 低于人物20%（人物推出画面） |
| **背景层次** | 背景层/中景层/前景层分离 |

## 七、特效类型说明（Kling生成 vs 后期合成）

> **重要**：下表中的特效类型，Kling只负责生成"基础画面"，带★的必须后期合成。

| 特效类型 | 处理方式 |
|---------|---------|
| 漫画文字弹出（「月薪三万」等） | ★ **后期合成** |
| 漫画箭头/思维泡泡 | ★ **后期合成** |
| 系统面板P04 | ★ **后期合成（独立图层）** |
| 蓝色光芒扩散（系统激活） | ★ **后期合成** |
| 字幕/标注文字 | ★ **后期合成** |
| 脑补闪回漫画边框 | ★ **后期合成** |
| 慢动作 | Kling参数设置（slow motion） |
| 虚化景深 | Kling可控（焦距/aperture参数） |
| 光晕/光斑 | Kling可生成（光线描述） |
| 颜色/色温 | Kling可控（色调描述词） |
| 唇印（P02瓶口） | Kling需明确提示词描述 |
