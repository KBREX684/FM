# 全局场景资产清单

> 本文件定义全剧所有场景的美术标准，所有集数共用。

## L01 · 长途大巴（车内）

| 属性 | 规格 |
|------|------|
| **类型** | 封闭空间 |
| **尺寸** | 标准45座长途大巴 |
| **色调** | 暖灰/旧米色为主，座椅为深蓝或深灰布面，轻微做旧 |
| **光线** | 前窗自然光（白天）→ 昏暗顶灯（夜晚），窗外来光为暖黄色调 |
| **关键元素** | 行李架（银灰色金属）、皮质/布质座椅、过道狭窄、车窗带淡蓝色贴膜 |
| **氛围** | 拥挤、封闭、略带压抑 |
| **使用频率** | 极高，第1-3集核心场景 |
| **制作优先级** | P0 |

## L02 · 魔都长途客运站（外景）

| 属性 | 规格 |
|------|------|
| **类型** | 半开放空间 |
| **色调** | 灰冷/阴天质感，加轻微噪点 |
| **光线** | 户外自然光（冬季，偏冷白），后期可通过后期调为暖黄 |
| **关键元素** | 人潮、大巴车队、路牌、客运站建筑（灰色矩形结构） |
| **氛围** | 春运的忙碌、拥挤、急切 |
| **使用频率** | 低，仅第1集开场 |
| **制作优先级** | P2 |

## 场景通用规范

- **线条**：场景线条比人物略简化，避免抢戏
- **上色**：场景整体饱和度低于人物20%，利用色彩将人物推出画面
- **景深**：背景层、中景层、前景层分层绘制，便于后期做虚化
- **比例**：人物与场景比例严格按7头身匹配
