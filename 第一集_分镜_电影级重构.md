## 一、如何阅读本文档

```
【单镜卡片结构说明】

镜头编号 · 景别 · 时长
├─ 机位  : 摄影机位置代码（如 CAM-A）
├─ 高度  : 摄影机垂直高度（俯角/平视/仰角）
├─ 焦距  : 等效全幅焦距（广角=强透视感，长焦=压缩空间感）
├─ 轴线  : 本镜处于180°轴线的哪一侧
├─ 方向  : 画面中主角运动/朝向方向（→左 / →右 / 面向画面）
├─ 画面  : 详细画面内容
├─ 对白  : 台词
├─ 音效  : 声音设计
├─ 情绪  : 情绪星级 + 功能说明
├─ 上镜连接 : 与上一镜的剪辑关系
└─ 下镜连接 : 与下一镜的剪辑关系（转场提示）
```

# 制作备注（电影级升级版）

### 01 · 竖屏9:16的镜头语言适配

> 与横屏电影不同，竖屏的景别重心在**垂直方向**。

- **近景/特写**：人物脸部可占画面上半部分，下半部分留衣服/道具——给道具叙事留空间
- **中景**：人物上半身+座位，座位信息辅助交代空间关系
- **远景**：利用大巴纵深（前后方向=画面上下方向），透视感更强
- **三人同框**：优先处理为正反打单人镜头组合，避免三人同时出现在9:16画面中导致人脸过小

### 02 · AI生图提示词框架（Kling）

每个镜头生成时建议包含以下结构：
```
[角色一致性锁定词] + [场景/机位描述] + [情绪/动作] + [色调/光线]

示例（S03-016白离震惊特写）：
"26岁中国男性，冷白皮，黑色碎发，桃花眼睁大，藏蓝修身西装+白衬衫，
白色入耳式耳机，大巴车内后排，正面特写，冷蓝色光芒照射在脸上，
瞳孔中反射蓝色数字面板光芒，震惊表情嘴微张，电影感，9:16竖屏"
```
