# 第一集 · 关键帧提示词文档（Keyframe Prompts）

> 用途：为每个镜头 / 子镜头生成 **Kling 首帧 / 尾帧参考图** 的提示词。每条都是 **现有素材 ID 的组合调用**，可直接复制到 MJ v6 / SDXL / FLUX 出图。  
> 命名规则：`KF-{镜号}` 默认为该镜头首帧；若需要锚定尾帧另行标注 `KF-{镜号}-end`。  
> 通用尾缀（每条均已附加）：`cinematic semi-realistic Chinese manhua, ultra-detailed face matching referenced asset, soft volumetric light, shallow depth of field, 8k photoreal, --ar 16:9`  
> 注意：**每条 prompt 中提到的"参考素材"字段即为 02 号文档中固化的资产 ID**，出图时务必把对应资产基图也一同作为图像参考喂入，以保证人物 / 服装 / 纹身 / UI 风格不漂移。
