## 三、Kling 3.0 制作备忘

1. **时长**：每个子镜头默认 ≤5s，超过的拆为多段；S08-02 类长镜可用首尾帧扩展。  
2. **运镜**：避免 360° 旋转、剧烈手持；Kling 对 PUSH / PULL / PAN / TILT / TRACK 稳定输出最好。  
3. **多角色同框**：超过 3 人画面建议使用首尾帧约束 + 关键帧锚定，参考 `04_关键帧提示词.md`。  
4. **一致性**：所有镜头 prompt 头部统一 prefix（详见 03 号文档），并强制引用资产 ID，保证人物长相、服装、场景灯光不漂移。  
5. **系统 UI**：蓝色面板统一用 `[PROP-PANEL-*]` 资产，建议后期合成而非直接生成，可减少不一致。  
6. **NSFW 风险**：S02-05b、S08-01c 仅暗示拥挤/卡通袜，画面收紧不越界。
